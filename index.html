We have designed the following based on the Stackoverflow Java dataset:

  1. Visualized the top packages that were discussed
  2. Visualized topics on a bubble chart based on text analysis
  3. Designed a bar graph to analyze monthly traffic of Questions, Answers and Accepted Answers.
  4. Designed a Sunburst for Weekly analysis of each topic/package. It provides helpful insights about when in the week you should be posting in order to maximize the probability of getting an answer
  5. A network visualization that maps popular users to topics and plots the time series data of votes gained.
  6. A SVM classification model for classifying the number of posts on day of week. While training, number of posts above median were assumed as High and below median were assumed as Low.
